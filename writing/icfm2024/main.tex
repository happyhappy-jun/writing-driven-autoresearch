% Provisional values. Both render unchanged; both are greppable in source.
%
%   \ph{}  = NO measurement backs this number. It is a written-ahead guess.
%            At the final audit: the real number lands, or THE SENTENCE IS DELETED.
%
%   \phm{} = MEASURED, but from a run that a same-protocol run is expected to
%            supersede (e.g. the 0.5B probe, pending the full R2 sweep).
%            At the final audit: UNWRAP IT. It is real and has a file behind it.
%            Deleting it would throw away a true, evidence-backed claim.
%
% The distinction exists because "delete anything still wrapped" is the right rule for
% one of these and catastrophically wrong for the other.
